identification Module¶

trackpy.identification.band_pass(img, p_rad, hwhm)¶

Intended to be a replacement for bpass in the matlab/IDL code.

Works by convolving a Gaussian with the image, than a box car and taking the difference.

Parameters:
  • img – array of data
  • p_rad – the size of the window used for the convolution
  • hwhm – the hwhm of the Gaussian
Return type:

numpy.ndarray scaled between 0 and 1
trackpy.identification.find_local_max(img, d_rad, threshold=1e-15)¶

This is effectively a replacement for pkfnd in the matlab/IDL code.

The output of this function is meant to be feed into subpixel_centroid()

The magic of numpy means this should work for any dimension data.

Parameters:
  • img – an ndarray representing the data to find the local maxes
  • d_rad – the radius of the dilation, the smallest possible spacing between local maximum
  • threshold – optional, voxels < threshold are ignored.
Return type:

(d,N) array of the local maximums.

trackpy.identification.subpixel_centroid(img, local_maxes, mask_rad)¶

This is effectively a replacement for cntrd in the matlab/IDL code.

Should work for any dimension data

Parameters:
  • img – the data
  • local_maxes – a (d,N) array with the location of the local maximums (as generated by find_local_max())
  • mask_rad – the radius of the mask used for the averaging.
